# BorderedTextField

The outlined text-field layout: the label sits inside a rounded chrome,
above the input. Used when a field's label should be visually integrated
with its control (e.g. dense form layouts).

`BorderedTextField` is a sibling to `TextField`, not a variant of it. The
two have meaningfully different APIs:

- `BorderedTextField` does **not** support `startSlot`.
- `endSlot`, when set, is rendered as a full-height side column. There is
  no inline `endSlot` position.
- `characterCountPosition` does not support `'inline'`. Use `'below'`
  (default) or `'hidden'`.

## Migration from `TextField variant="bordered"`

Replace `<TextField variant="bordered" .../>` with
`<BorderedTextField .../>`. Drop any `startSlot`, `endSlotPosition`, or
`characterCountPosition="inline"` props — those are no longer supported in
the outlined layout. If you need a leading icon, consider using `TextField`
(default) instead. If you need a side action, use `endSlot`.
