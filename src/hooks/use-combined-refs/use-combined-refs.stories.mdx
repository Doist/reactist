# useCombinedRefs

`useCombinedRefs` allows you to merge a passed down from above and your own ref together into one reference.

```TypeScript
import { useCombinedRefs } from '@doist/reactist'

function MyComponent(props, ref) {
    const myRef = useRef()
    const combinedRef = useCombinedRefs(ref, myRef)

    return (
        <Box ref={combinedRef} />
    )
}
```
